The diff gate compares each new frame against the previous one at 64×64, summing per-channel absolute differences. Frames under the 5% threshold are dropped before encode — the cheapest API call is the one you don't make. Adaptive cadence reads recent activity and either slows down for an idle student, speeds up for an active one, or kicks closer when nothing has moved for a minute.
